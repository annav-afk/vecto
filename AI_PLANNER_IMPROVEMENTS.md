# Улучшения AI-планировщика Vecto

## Что изменилось

### ✨ Реалистичная генерация планов

AI-планировщик Vecto теперь генерирует **реальные, выполнимые планы** на основе ваших временных ограничений.

### 🎯 Ключевые улучшения

#### 1. **Расчёт доступного времени**
- AI теперь рассчитывает общее доступное время: `дни × часы в день`
- Учитывает ваш дедлайн и количество часов в неделю
- Пример: 90 дней × 1.4 часа/день = 128 часов доступно

#### 2. **Конкретные задачи**
Больше никаких абстрактных задач! 

❌ **Раньше:**
- "Работать над дизайном"
- "Заняться маркетингом"
- "Написать контент"

✅ **Теперь:**
- "Создать мокапы главной страницы и 3 основных экранов — 8 часов"
- "Провести интервью с 3 пользователями — 6 часов"
- "Написать 5 страниц контента для лендинга — 4 часа"

#### 3. **Реалистичные временные оценки**
- Каждая задача имеет конкретную оценку времени
- Сумма всех задач не превышает доступное время более чем на 20%
- Задачи распределены равномерно по этапам

#### 4. **Маленькие, выполнимые шаги**
- Задачи разбиты на конкретные действия
- Каждая задача содержит 2-4 подзадачи
- Можно выполнить за одну сессию работы

### 📊 Визуальные индикаторы

#### На странице создания плана:
- Показывается расчёт доступного времени в реальном времени
- Пример: "AI рассчитает план на **90 дней** с доступными **128 часами**"

#### На странице плана:
- Блок "План сгенерирован AI" с информацией:
  - Часов в неделю: 10ч/нед.
  - Общая длительность: 90 дней
  - Доступное время: 128ч

### 🔧 Технические детали

#### Промпт AI
Обновлённый промпт инструктирует AI:
1. Рассчитывать общее доступное время
2. Разбивать цель на логические этапы (фазы)
3. Создавать конкретные, выполнимые задачи
4. Не превышать доступное время
5. Делать задачи маленькими и реалистичными

#### Структура плана
```typescript
interface Plan {
  id: string;
  goal: string;
  deadline: string;
  hours_per_week: number;  // ← Используется AI
  total_days: number;       // ← Рассчитывается AI
  phases: Phase[];
  created_at: string;
}
```

### 🚀 Как это работает

1. **Пользователь вводит:**
   - Цель: "Запустить SaaS-продукт"
   - Дедлайн: через 3 месяца
   - Часов в неделю: 10ч

2. **AI рассчитывает:**
   - Всего дней: 90
   - Часов в день: 10 / 7 = 1.4ч
   - Всего доступно: 128 часов

3. **AI генерирует:**
   - 4-5 этапов (фаз)
   - 3-5 задач в каждой фазе
   - Каждая задача с конкретным временем
   - Сумма всех задач ≈ 128 часов (±20%)

4. **Результат:**
   - Реалистичный, выполнимый план
   - Конкретные действия с временными оценками
   - Распределение по датам

### 📝 Примеры конкретных задач

#### Для разработки:
- ❌ "Разработать бэкенд"
- ✅ "Создать REST API для авторизации пользователей — 6 часов"

#### Для маркетинга:
- ❌ "Настроить рекламу"
- ✅ "Запустить 3 рекламные кампании в Facebook Ads — 4 часа"

#### Для контента:
- ❌ "Написать статьи"
- ✅ "Написать 5 SEO-статей по 1000 слов каждая — 10 часов"

### 🎓 Правила генерации

AI следует строгим правилам:
- **НЕТ** абстрактных задач
- **НЕТ** мотивационных текстов
- **НЕТ** пояснений в названиях
- **ТОЛЬКО** конкретные действия + время
- **ТОЛЬКО** маленькие, выполнимые задачи
- **ТОЛЬКО** реалистичные оценки

### 🔄 Фолбек (fallback)

Если AI недоступен, система автоматически использует локальный mock-генератор с шаблонами для разных типов целей (продукт, обучение, фитнес и т.д.).
